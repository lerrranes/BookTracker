# Movie Library — Личная кинотека

**Автор:** Валерия Витальевна Неснова
**Дата:** Май 2026

## Описание программы
Приложение «Личная кинотека» предназначено для хранения списка ваших любимых фильмов. Программа позволяет добавлять новые записи, хранить их в формате JSON и фильтровать список по жанрам или годам выпуска.

## Функционал
- Добавление фильма (Название, Жанр, Год выпуска, Рейтинг).
- Валидация данных: проверка на пустые поля, корректность года и диапазон рейтинга (0-10).
- Сохранение и загрузка данных из файла `movies.json`.
- Фильтрация по жанру и году выпуска.
