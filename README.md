# Конвертер валют с поддержкой API

**Автор:** Валерия Неснова
**Дата создания:** Май 2026

## Описание программы
Приложение позволяет конвертировать валюты в реальном времени, используя данные сервиса ExchangeRate-API. Включает в себя выбор валют, ввод суммы и сохранение истории операций в JSON-файл.

## Инструкция по получению API-ключа
1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Зарегистрируйтесь и подтвердите email.
3. Скопируйте ваш "API Key" и вставьте его в переменную `API_KEY` в файле `main.py`.

## Инструкция по запуску
1. Установите библиотеку requests: `pip install requests`
2. Запустите файл: `python main.py`

## Функционал
- Получение актуальных курсов через внешний API.
- Проверка ошибок (пустые поля, ошибки сети, неверные данные).
- Автоматическое сохранение истории в `history.json`.
- Просмотр истории в таблице внутри приложения.
