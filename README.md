## Описание программы

Приложение для учета прочитанных книг. Позволяет пользователю сохранять информацию о книгах и быстро фильтровать список по заданным критериям. Данные сохраняются локально в формате JSON.

### Возможности:
- Добавление книг с указанием названия, автора, жанра и количества страниц
- Фильтрация книг по жанру (Роман, Фантастика, Детектив, и др.)
- Фильтрация по количеству страниц (>200, >300, >500)
- Сохранение данных в JSON-файл
- Зарузка данных из JSON-файла
- Удаление выбранной книги
- Очистка всего списка книг
- Валидация вводимых данных (проверка пустых полей, числовое значение страниц)

## Инструкция по запуску

1. Установите Python 3.
2. Скачайте файл `main.py`.
3. Запустите его командой: `python main.py`

### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter (обычно идет в комплекте с Python)

## Примеры использования
- Добавление: Введите название "Преступление и наказание", автор "Ф. Достоевский", жанр "Классика", страницы "600". Нажмите "Добавить".
- Поиск: Введите в фильтр "Классика" для отображения только этого жанра.
- Фильтр 200+: Нажмите кнопку "Страниц > 200", чтобы увидеть только толстые книги.
